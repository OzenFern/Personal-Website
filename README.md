# Personal Portfolio Website

A professional, fully responsive personal portfolio website built to showcase my projects, technical skills, and ongoing learning in Software Engineering.

**Live Demo**: [_Click Here_](https://ozenfern.github.io/Personal-Website)

## Overview

As a Computer Science & Engineering student, I built this portfolio from the ground up using core web technologies (HTML and vanilla CSS). My goal was to create a sleek, dark-themed UI that relies on modern CSS architecture rather than heavy CSS frameworks.

The site features a custom design system, glassmorphism elements, and smooth scroll animations to create an engaging user experience.

## Key Features

- **Modern Dark Theme UI:** Built using a custom color palette and typography system (Inria Serif, Inter, and Bricolage Grotesque).
- **Glassmorphism Design:** Utilizes advanced CSS `backdrop-filter` and `color-mix()` properties to create sleek, frosted-glass project cards and navigation panels.
- **Fully Responsive Layout:** Implements CSS Grid and Flexbox for fluid adaptability across desktop, tablet, and mobile devices, including a custom JavaScript-free slide-out hamburger menu.
- **Dynamic Animations:** Integrates the AOS (Animate On Scroll) library for smooth, staggered entrance animations, alongside custom CSS hover states and transitions.

## Tech Stack

- **Structure:** Semantic HTML5
- **Styling:** CSS3 (Custom Variables, CSS Grid, Flexbox, Pseudo-elements, Media Queries)
- **Interactivity:** CSS Checkbox Hack (for mobile menu)
- **Animation Library:** [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)

## Sections

1. **Hero/Home:** Quick introduction with direct links to my Resume and GitHub.
2. **About:** A brief overview of my background, interests in AI/ML, and tech journey.
3. **Skills:** Categorized technical proficiencies (Programming Languages, Web Dev, Tools).
4. **Projects:** Detailed grid layout of recent work (Flooder Game, Powerball Simulator) featuring tech stacks and repository links.
5. **Contact:** A minimalist connection section directing visitors to my professional GitHub and LinkedIn networks.

## How to Run Locally

Since this project is built with static files, no build steps or package managers are required.

1. Clone the repository:

   ```bash
   git clone https://github.com/OzenFern/Personal-Website
   ```

2. Navigate to the project directory:

   ```bash
   cd Personal-Website
   ```

3. Open `index.html` in your preferred web browser. (Alternatively, use an extension like VS Code Live Server for hot-reloading).
